# Kanban To-Do List App

## Description

A fully interactive Kanban-style to-do list application built with HTML, CSS, and JavaScript.

## Features

- Drag & Drop task management
- Due date & live countdown
- Priority-based color system
- LocalStorage persistence
- Task repetition (daily / weekly)
- Topic tagging & filtering
- Recycling bin with task history
- Responsive design

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## How to Run

Open `index.html` in your browser  
OR deploy via GitHub Pages.
